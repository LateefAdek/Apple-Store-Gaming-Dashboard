# Apple-Store-Gaming-Dashboard
# Project Title: Apple Store Application Analysis
Project Overview: This project features a dynamic business intelligence solution designed to analyze the competitive landscape of mobile gaming on the iOS platform. By synthesizing raw App Store metadata, the dashboard provides a high-level overview of user sentiment (ratings), demographic accessibility (age ratings), and global localization trends. It serves as a strategic tool for identifying high-growth opportunities and evaluating the effectiveness of diverse monetization models across yearly cycles.

# Tech Stack:This dashboard was built with the following technologies
Tools: PowerBi, PowerQuery, DAX.
Visualisation: Interactive slicers, Star Schema Modelling, Bar, Line, and Column chart.
Data Cleaning: ETL processes and Data normalisation

# Key Insights
1. In the year 2009,they has the highest revenue of 1.11M, with an average pricing of $0.91, average rating of 3.67 for a total of 175 games.
2. In 2015, they had the highest total of games of 2190 with average user rating of 4.50, but yet a lower revenue generation of $0.25 due to the large ratio of free games to paid games.
3. The top 5 rated application are games which are free and mainly for kids above 9 years.
4. The most used language in the applications is English.
